Open Source Projects

PyQOptics

Python Quantum Optics (PyQOptics) is a full-stack quantum simulation Python library which can be used to simulate continuous variable quantum circuits in the Fock or Gaussian backend.

With a focus on the preparation methods of approximate Gottesman-Kitaev-Preskill (GKP) states, one can model the impact of imperfect sources by including the spectral degrees of freedom, as well as interpret the state's loss tolerance by quantifying the effective squeezing.

News Classifier

A simple machine learning project written in Python that classifies news articles or headlines as real or fake using natural language processing and supervised learning. It utilises pandas for data handling and implements ML methods using scikit-learn.

NOTE: This is not a perfect model, and works best for U.S. politics headlines, as it is trained on such data.
